# Arduino Radar System

## Overview
Arduino Radar System is an object detection project built using an Arduino Nano, HC-SR04 Ultrasonic Sensor, and SG90 Servo Motor.

The system scans its surroundings, detects nearby objects, and displays their position and distance through a radar-style visualization.

## Features

- Real-time object detection
- Ultrasonic distance measurement
- Servo-based scanning mechanism
- Radar GUI visualization
- Hardware and simulation implementation

## Components Used

- Arduino Nano
- HC-SR04 Ultrasonic Sensor
- SG90 Servo Motor
- Buzzer
- Breadboard
- Jumper Wires

## Working Principle

The servo motor rotates the ultrasonic sensor from one side to another.

The HC-SR04 measures object distance and sends data to the Arduino.

The detected object position and distance are displayed on a radar-style graphical interface.
